# Build instruction

## Windows

1. Clone this repo
2. run `go mod download`
3. run the `build.bat` script

## Others

1. clone this repo
2. run `go mod download`
3. build normally, but version menu won't show the correct information.  
   
If you have made a script to build with correct build flags for other platform please do a pull request.
